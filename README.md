# RLab - Building Todo List 
- Created a functional streamlined Todo List using React/Javascript that features input box and a default tasks list. When a new task is added, it defaults to add to the top of the list returning the new object before the ...state on the addTask case. 

- On each task there are edit and delete buttons. The delete button is disabled until the task is toggled to complete using the checkbox. There is also a reset button that is supposed to reset the list back to the default tasks. 

- I first created an app without the save and disabled deleted function for a more streamlined approach then updated the app and commented out that previous code to use as a boilerplate for future projects. 

- This is a very useful application especially if you are an organized person and want to keep up with your tasks


## Technologies Used 
- React 
- JavaScript 
- useReducer
- useState
- Vite
